# Portfolio Copy

## GitHub repo description

AI invoice exception handler MVP with deterministic rules, human-in-the-loop AI fix proposals, and a learning loop for AP teams.

## Short portfolio blurb

Built an AI invoice exception handler MVP for accounts payable teams. The product uses deterministic rules to detect issues, an LLM to propose fixes, human review for financial judgment, and a learning loop that saves approved resolutions for future exceptions.

## Slightly longer portfolio blurb

Designed and built an AI invoice exception handler MVP focused on the part of AP workflows where teams lose the most time: exceptions. The system detects issues with deterministic rules, generates structured fix proposals with an LLM, keeps humans in the loop for approval, and learns approved resolutions so repeated exception patterns become faster to handle over time.

## PM-angle blurb

This project was built as an MVP case study to show product thinking, not just implementation. It focuses on workflow design, safety boundaries, business metrics, and the compounding value of a human-in-the-loop learning system for finance operations.
